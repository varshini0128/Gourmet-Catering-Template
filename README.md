# Gourmet au Catering Website

A clean, minimalist, and fully responsive single-page website template designed for catering businesses and restaurants. This project focuses on elegant typography, high-quality imagery, and a seamless user experience across all devices.

## Features

  * **Responsive Design:** Optimized for mobile, tablet, and desktop views using CSS Media Queries.
  * **Sticky Navigation:** Quick access to site sections (About, Menu, Contact) at all times.
  * **Dynamic Layout:** Alternating image and text sections using CSS Flexbox.
  * **Catering Menu:** Beautifully formatted menu list with pricing and descriptions.
  * **Booking Form:** A stylized contact and reservation form.
  * **Semantic HTML:** Built with SEO and accessibility best practices.

## Technologies Used

  * **HTML5:** Structural layout and semantic elements.
  * **CSS3:** Custom styling, Flexbox layouts, and responsive design.
  * **Google Fonts/Web Safe Fonts:** Elegant serif typography for a premium feel.

## Getting Started

### Prerequisites

You only need a modern web browser (Chrome, Firefox, Safari, or Edge) to view this project.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/varshini0128/Gourmet-Catering-Template.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd gourmet-catering
    ```
3.  Open `index.html` in your browser.

## Project Structure

  * `index.html` - The main structure of the website.
  * `style.css` - Custom styles and responsiveness rules.
  

## Usage

This template is easily customizable. To update the content:

  * Edit the `index.html` file to change the business name, menu items, or contact details.
  * Modify `style.css` to adjust the color scheme or typography to match your brand.
